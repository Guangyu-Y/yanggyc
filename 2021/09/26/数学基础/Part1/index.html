2、数据类型(分类)

2.1

  1. 离散型（N）
  2. 稠密型（Q）
  3. 连续型（R）

    一个实数轴中，取一个无限小的 区间，可能不包含一个数据集的元素。则这个数据集为离散型。若这个小区间无论取多小，一定包含这个数据集的元素，则这个数据集是稠密的。如自然数是离散的，有理数是稠密的，但不是连续的。实数是稠密、连续的。一般情况下直接分为离散型与连续性数据。

  • 离散型数据：指一个只取有限个或可数无限个数值的随机变量。
  • 连续型数据：指一个取任何实数的概率都为0的变量。

2.2

  • 横截面数据：一个时间点、段内的所有数据。如：上个月的销量。
  • 时间序列数据：考虑进时间属性的数据。如：股票的市值随时间的变动。去年的销量。
  • 面板数据：时间属性 + 空间属性的数据。去年一年所有省市的销量数据。

2.3

  • 定类尺度（无序）如，国籍：中国
    • 不可排序
    • 不可运算
  • 定序尺度（有序）如，健康状况：良好
    • 可排序
    • 不可运算
  • 【数值】定距尺度（0点有意义）如，出身年份：1981
    • 0点有特定意义
    • 可排序、可运算
  • 【数值】定比尺度（0点表示不存在）如，体重：56公斤
    • 可排序，可运算
    • 0点无特定意义

3、数学模型

  1. 数学分析模型

    • 传统统计模型（解决结构问题） -> 回归模型 -> 计量
    • 数据挖掘模型（解决预测模型）-> 决策树、神经网络、支持向量机、贝叶斯网络、关联规则…

  2. 传统回归模型

    • 最简单的回归模型是一元线性回归模型

    $$
    Y = \beta_0 + \beta_1X + \epsilon
    $$

    • 扩展到多个自变量的形式，多元线性回归模型
      $$
      Y = \beta_0 + \beta_2X_2 + \beta_3X_3 + … + \beta_pX_p + \epsilon
      $$
    • 非线性形式，广义回归模型
      $$
      Y = f(X_1,X_2,…,X_3) + \epsilon
      $$

  3. 常用的回归模型

    1. 横截面模型：多元回归、逻辑回归、托宾回归【0膨胀问题(用户0过多，影响模型训练)】、截尾回归
    2. 时间序列模型：ARIMA(一次只能分析一个时间序列)、HARCH（分析时间序列里的异方差）、协整（多个稳定的时间序列）
    3. 面板数据模型：固定效应/随机效应、空间计量模型
    4. 结构方程模型：（多个多元回归拼成方程组，多个有关系的Y拼成方程组）
    5. 向量自回归模型（VAR）：（Y也对X有影响）
