1、callable()

检查对象是否可调用。如果返回True，object仍然可能调用失败，但若返回False，一定不能调用成功。

2、__getattribute__

属性访问拦截器，就是当这个类的属性被访问时，会自动调用类的__getattribute__方法。Python中只要定义了继承object的类，就默认存在属性拦截器，只不过是拦截后没有进行任何操作，而是直接返回。所以我们可以自己改写__getattribute__方法来实现相关功能，比如查看权限、打印log日志等。
